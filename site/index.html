Проектная деятельность

Cтудент группы: 243-332

Трофимов И.А.

Цель практики

Отработка навыков создания конечного продукта, для потребителя. Построение проекта от начальных этапов, до компиляции готового приложения.

Шаги

Описание этапов создания проекта

Этап 1

Идея продукта

Создание базовой идеи для макетирования продукта. Разработка миссии проекта. Разработка ТЗ. Первичныне наброски того, как будет выглядеть приложение

Этап 2

Написание программного кода

Создание программного кода приложения на HTML. Разработка меню, в соответствии с предпологаемым функционалом, согласно техническому заданию проекта

Этап 3

Макетирование и дизайн

Начало работ по сборке модели внешнего вида приложения, на основе созданного ранее кода. Создание и тестирование кнопочного функционала, пунктов меню и навигации в приложении. Подготовка логотипов и иконок для кнопок управления

Этап 4

Финальная сборка приложения

После завершение всех мероприятий по тестированию и отладке функций приложения, осуществляется его финальная сборка из кода, для получения готового продукта. Размещение программы в App Store и Google Play

Журнал проекта

  1. 6 Июня 2025 г: Начальный этап выполнения практического задания. Определние целей и задач. Поиск проекта, для реализации

  2. 7-8 Июня 2025 г: Создание git-репозитория и подготовка структуры сайта проекта. Принято решение делать мобильное приложение - поиск темы для мобильного приложения

  3. 9 Июня 2025 г: Тема для мобильного приложения: трекер эклогичного образа жизни с возможностью ослеживать углеродный след и точки сбора мусора, годного к переработке

  4. 10-11 Июня 2025 г: Начата подготовка кода приложения в HTML, отладка и сборка начальных, тестовых версий программы. Работы по доводке и изменнию дизайна

  5. 12 Июня 2025 г: Финальная сборка программного кода и выгрузка готового приложения, для публикации. Начало подготовки материалов для отчетности по практике
